# CodeAlpha Internship - Hangman Game

## Description
This project is a simple Hangman game built using Python. The user guesses letters to find the hidden word within limited attempts.

## Features
- Random word selection
- Tracks guessed letters
- Limited attempts
- Win/Lose condition

## Technology Used
- Python

## How to Run
1. Open terminal
2. Navigate to file location
3. Run:
   python hangman.py
